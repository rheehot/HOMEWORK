### W3C에서 표준으로 인정하는 step
d > wd > cr > pr > r

Ab­bre­vi­a­tion	Full name
1. FPWD    - First Public Working Draft
2. WD      - Working Draft
3. CR      - Candidate Recommendation
4. PR      - Proposed Recommendation
5. REC     - Recommendation
6. SPSD    - Superseded Recommendation
